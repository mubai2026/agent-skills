# Continuous Learning · 持续学习

每次会话结束后自动评估并提取可复用的模式，存入技能库供未来使用。

## 学习流程

1. **会话评估**：会话结束时检查是否有值得学习的模式
2. **提炼直觉**：将模式提炼为原子化的"直觉"（低粒度知识块）
3. **置信度评分**：相同模式出现多次后提升置信度
4. **技能生成**：高置信度的模式集群自动生成新的 SKILL.md
5. **入库**：新技能自动加入仓库，所有人可用
